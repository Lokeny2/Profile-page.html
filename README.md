    Personal Profile Website
A personal portfolio website built with Semantic HTML5. This project highlights technical skills, linguistic proficiencies, 
and key development projects.

    Features
1.Semantic Structure: Built using modern HTML5 elements for better SEO and accessibility.

2.Multilingual Showcase: Dedicated section highlighting proficiency in French (B2), Chinese (HSK 3 candidate), and English.

3.Project Spotlight: Features ShambaSmart AI, a specialized agricultural digital platform.

4.Responsive Contact Info: Integrated mailto links and location data for easy professional networking.

    Technologies Used
1.HTML5: For content structure and semantics.

2.Git/GitHub: For version control.

    Lessons Learnt
During the development of this profile page, I focused on:
    1.Accessibility : Ensuring that the use of semantic tags allows screen readers to navigate the professional summary and 
    skills efficiently.
    2.Scalable Content: Structuring the "Key Projects" and "Technical Proficiencies" using <article> tags to make it easy for me 
    to add new milestones in the future.
    3.Cross-Domain Integration: Learning how to effectively present a diverse background (Language Specialist + Software Developer) 
    in an eye appealing manner.
